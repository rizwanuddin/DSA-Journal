# 📓 DSA Journal

A personal journal to track my progress through Data Structures and Algorithms — solved in **Python** and **Java**.

---

## 🎯 What is this?

This is my hands-on DSA study log. Every problem I solve, every concept I learn — it lives here. The goal is to go from zero to solid DSA fundamentals, stay consistent, and have a clean record of my progress.

I'm using this repo across two laptops, synced via Git, so everything stays in one place.

---

## 📚 Course I'm Following

**[Striver's A2Z DSA Course/Sheet](https://takeuforward.org/dsa/strivers-a2z-sheet-learn-dsa-a-to-z)** by [takeUforward](https://takeuforward.org)

This is a free, structured DSA course that covers **474 problems** going from absolute basics all the way to hard-level topics. It's organized step by step — you don't jump around, you build on top of what you already know.

### Topics covered (in order):
1. Basics (patterns, recursion fundamentals)
2. Sorting Techniques
3. Arrays (Easy → Medium → Hard)
4. Binary Search
5. Strings
6. Linked Lists
7. Recursion
8. Bit Manipulation
9. Stacks & Queues
10. Sliding Window & Two Pointers
11. Heaps
12. Greedy Algorithms
13. Binary Trees
14. Binary Search Trees
15. Graphs
16. Dynamic Programming
17. Tries

I'm solving each problem in **Python** and/or **Java**, writing notes on what I learned, and pushing everything here.

## ⚙️ Setup

If you're cloning this or setting it up on a new machine:

```bash
git clone https://github.com/rizwanuddin/DSA-Journal.git
cd DSA-Journal
code .
```

Daily workflow:
```bash
git pull          # before you start
git add .
git commit -m "what you did"
git push          # after you're done
```

See `cheatsheet.md` for Git commands and VS Code shortcuts.
